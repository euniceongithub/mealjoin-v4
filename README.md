# MealJoin Frontend

MealJoin is a modern web application that connects people through shared home-cooked meals. This frontend was built with React + Vite, offering a clean, responsive, and user-friendly interface for discovering and joining meal experiences within your community.

## Getting Started
Prerequisites

Before you begin, ensure you have the following installed:

Node.js
 (v16 or higher)

npm (comes with Node.js)

## Installation

Clone the repository and install dependencies:

git clone https://github.com/<your-username>/mealjoin.git
cd mealjoin
npm install

## Development

To start the development server:

npm run dev


After a few seconds, open your browser and navigate to:

http://localhost:5173

Build for Production

When you’re ready to deploy:

npm run build


This will generate optimized static assets in the dist/ directory.

Tech Stack

React (TypeScript) — Component-based UI

Vite — Fast build tool and dev server

Tailwind CSS + shadcn/ui — Modern styling and UI components

React Router — Client-side routing

Axios / Fetch API — Backend integration


## Developer Notes

Ensure environment variables (e.g., backend API URLs) are correctly set in a .env file.

Follow the project’s existing design language for consistency.

Run npm run lint before pushing code to maintain code quality.
